# DrRisadinha

"Fale com o Dr. Risadinha" é um blog de informações sobre a saúde da criança e do adolescente mantido por uma equipe de pesquisa ligada ao HC Criança de Ribeirão Preto, SP. 

Este projeto procura extrair e estruturar a informação apresentada nesse blog, disponibilizando-a em formato apropriado para aplicações na Web Semântica e incorporação à nuvem de dados abertos conectados.
